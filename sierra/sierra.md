# Documentación Técnica - Tarjeta Digital

## Decisiones Semánticas

Para realizar esta página utilicé etiquetas semánticas para organizar mejor el contenido:

- Usé <header> para el encabezado con mi nombre.
- Usé <nav> para el menú de navegación.
- Usé <main> para el contenido principal.
- Usé <section> para dividir la información (sobre mí y habilidades).
- Usé <article> dentro de habilidades para agrupar información relacionada.
- Usé <aside> para contenido adicional.
- Usé <footer> y <address> para el contacto.

Esto ayuda a que la página sea más clara y fácil de entender.

## Árbol DOM

La estructura de la página es:

html
 └── body
      ├── header
      │    └── nav
      ├── main
      │    ├── section
      │    └── section
      │         └── article
      ├── aside
      └── footer

## Conclusión

El uso de HTML semántico permite organizar mejor el contenido, mejorar la accesibilidad
y facilitar que los buscadores entiendan la página.
