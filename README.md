# once--javiera
<h2 style="text-align:center; color:purple; text-decoration:underline;">
 Cómo creé mi repositorio en GitHub 
</h2>

<p><strong>Pasos que seguí para subir mi pagina a GitHub:</strong></p>

<ul>
  <li><strong>1.</strong> Entre a la pagina de GitHub e inicie sesión con mi cuenta.</li>
  <li><strong>2.</strong> Hice clic en el botón <mark>New repository</mark> para crear un repositorio nuevo.</li>
  <li><strong>3.</strong> Escribí el nombre de mi repositorio: <strong>once-javiera</strong>.</li>
  <li><strong>4.</strong> Seleccioné la opción <mark>Public</mark> para que mi página pueda verse en internet.</li>
  <li><strong>5.</strong> Hice clic en <strong>Create repository</strong>.</li>
  <li><strong>6.</strong> Luego presioné <mark>Add file</mark> y despues <mark>Create new file</mark>.</li>
  <li><strong>7.</strong> Creé un archivo llamado <strong>index.html</strong>.</li>
  <li><strong>8.</strong> Copié y pegué mi código HTML dentro del archivo.</li>
  <li><strong>9.</strong> Finalmente presioné <strong>Commit changes</strong> para guardar los cambios.</li>
</ul>

<p style="text-align:center;">
<strong>Así pude subir mi página web a GitHub paso a paso.</strong>
</p>
