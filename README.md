<h2 style="text-align:center; color:purple; text-decoration:underline;">
 Cómo subí mi página web a GitHub 
</h2>

<p><strong>Pasos que realicé para publicar mi página en GitHub:</strong></p>

<ul>
  <li><strong>1.</strong> Ingresé a la página oficial de GitHub y abrí sesión con mi usuario.</li>
  <li><strong>2.</strong> Hice clic en el botón <mark>New repository</mark> para crear un repositorio.</li>
  <li><strong>3.</strong> Escribí el nombre de mi repositorio: <strong>pagina-valentina</strong>.</li>
  <li><strong>4.</strong> Elegí la opción <mark>Public</mark> para que cualquier persona pueda ver mi página en internet.</li>
  <li><strong>5.</strong> Después di clic en <strong>Create repository</strong>.</li>
  <li><strong>6.</strong> Luego seleccioné <mark>Add file</mark> y después <mark>Create new file</mark>.</li>
  <li><strong>7.</strong> Creé un archivo llamado <strong>index.html</strong>.</li>
  <li><strong>8.</strong> Pegué mi código HTML dentro del archivo.</li>
  <li><strong>9.</strong> Finalmente hice clic en <strong>Commit changes</strong> para guardar todo.</li>
</ul>

<p style="text-align:center;">
 <strong>De esta forma logré publicar mi página web en GitHub paso a paso.</strong>
</p>
